Retreat Planner AI 

A state-driven chatbot designed for a wellness retreat platform.
It helps users plan retreats or explore individual healing sessions through a guided, conversational experience.



Features:

* Guided flow (retreat vs sessions)
* Retreat recommendation (2, 5, 7 days)
* Day-wise itinerary with timeline UI
* Individual session exploration ($500 / session)
* Shuffling FAQ support + contact details
* Compact widget + expandable view


Tech Stack:

* React + TypeScript
* Tailwind CSS
* Framer Motion
* Vercel (deployment)


Approach:

Built as a **state-driven chatbot** instead of a generic AI chat.
Uses a structured knowledge base and guided flows to provide clear, reliable responses.



Why I built this:
I wanted to build something that would actually help somepone or some business , not a generic chatbot so when I was looking for products, I found a website in which I had to look for various plans and various services they provide by going very deep into the website so I thought the AI chatbot can help reduce this time and just display all options,services and basic FAQ's and can also schedule meetings or bookings.


Run Locally:

npm install
npm run dev
